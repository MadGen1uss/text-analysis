1. Как разбить строку на список слов?
В Python для этого чаще всего используется метод строк .split().

Как это работает: Метод сканирует строку и разделяет её на части везде, где встречается разделитель (по умолчанию — любой пробельный символ: пробел, табуляция или перенос строки). Результатом работы метода всегда является список (list) строк.
2. Как подсчитать количество элементов?
Для подсчета количества элементов в любой последовательности (списке, строке, словаре, множестве) используется встроенная функция len() (сокращение от length — длина).

Как это работает: Функция возвращает целое число, соответствующее количеству объектов в контейнере.
